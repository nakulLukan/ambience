# Ambience (addon)

Connect kodi video to 'Ambience' backend. Ambience projects with dynamic lighting.

============
Installation
============

1. Launch Kodi >> Add-ons >> Install from zip file >> select 'script.ambience.matrix.zip'.
2. Start the addon from 'Programs add-ons'.
3. Play some video in kodi and you can see the dynamic lighting projected by the abience backend module.
